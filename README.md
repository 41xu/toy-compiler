# toy_js_compiler
## 项目结构

```
src
├── main.cpp
├── error.cpp
├── error.hpp
├── lexical_analyzer.cpp
├── lexical_analyzer.hpp
├── token.cpp
└── token.hpp
```



1. `main.cpp`

   主文件，处理命令行参数。

2. `error.xpp`

   错误类，用于报错提示。

3. `token.xpp`

   Token类，用于记录Token。

4. `lexical_analyzer.xpp`

   语法分析器类，用于语法分析
